JNOUS.COM

NAME
       jnous - variance-lab findings from empirical local LLM measurement

DESCRIPTION
       17 findings from 105,000+ inferences across 28 models. Each finding states
       what was tested, what was measured, and what the data showed. No theory
       without data. No data without method.

FINDINGS

       01. Three Questions for Agentic Autonomy
           Universal intake: human boundary, autonomy blockers, subtractive access.

       02. Delegation vs Inline Execution
           Delegated parallel: 48% wall clock of single-context. No context contamination.

       03. Interaction Mode Variance
           Passenger mode: 41x tokens vs governor mode. 88 sessions, single operator.

       04. Delegated Agent Authorization Gap
           Agents fail at auth boundaries, not capability boundaries.

       05. Governance Binding
           Governance binds at N=30: 81% overall. Reflex-dependent. Delivery-path sensitive.

       06. Interaction Mode Rerun
           52.7x passenger/governor under pre-committed rubric. N=501 qualifying sessions.

       07. HTTP/2 vs HTTP/1.1
           HTTP/2 multiplexing: 2.1x throughput for local inference concurrency.

       08. 1-Bit Quantization
           1-bit breaks the 8GB deployment ceiling. Bonsai 8B on 8GB M2 Mini.

       09. 1-Bit Across Hardware Tiers
           1-bit wins across 4 tiers. The bottleneck differs per tier.

       10. Throughput Ceiling
           Aggregate inference plateaus at a hardware-determined ceiling.

       11. Review vs Verification
           Cheaper model finds crash path expensive model misses.

       12. Lookdown Routing
           grep beats inference for known-answer retrieval.

       13. Manifest vs BM25
           Manifest-routed retrieval beats BM25 for small-corpus QA.

       14. Governance Refusal
           In-vivo reproduction of finding 5 during scorer calibration.

       15. Reflex Binding
           Abstract reflexes transfer only with lineage, not instruction.

       16. Effort-Dependent Binding
           Governance binding varies by effort tier. 4 models, 1872 trials.

       17. Handler Substrate Selection
           Three-gate methodology for substrate-code model selection.
