SEARCHING SKILLS

Ask a "how do I" question:
- "how do I connect to wifi"
- "teach me to check storage"
- "steps to shut down safely"

Browse by domain:
- Type: skills
- Type: skills cooking (to list one domain)

Show a specific skill:
- Type: show 1 (after a search returns a list)

Add your own:
- Type: skill add cooking/boil-eggs
- Edit the file that opens. Save. Done.

Notes:
- Skills are plain text files in ~/.subtract/skills/.
- Edit them directly with any text editor.
- Kiwix answers factual questions (what is, who was).
- Skills answer procedural questions (how do I, steps to).
