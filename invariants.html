# invariants.txt

The buffer-subtract pattern. Subtract's interface contract across modalities.

## The six invariants

A substrate honors the pattern if and only if all six hold:

1. **Native modality owns the buffer.** Subtract never owns it. Existing editing affordances (cursor keys, backspace-tile, "scratch that") work unchanged.

2. **Subtraction is explicit.** A specific gesture (keystroke, tile, wake phrase, HTTP call) hands the buffer to subtract. No polling, no ambient observation, no keystroke-by-keystroke.

3. **Subtract reads, translates, writes back.** It does not commit. The user's commit gesture (enter, speak-tile, endpoint) remains the only path to action.

4. **Non-subtraction has zero residue.** A user who never invokes subtract has a substrate bit-identical to one without subtract installed.

5. **Successful subtractions train toward direct expression.** The translation is visible. Over time, users internalize the direct form and stop subtracting.

6. **The router is modality-blind.** String in, string out. Same engine handles all substrates. Modality enters via which corpus and which trigger, not the router itself.

## The required precondition

The buffer must be inspectable without committing.

Modalities where intent commits as it accumulates (streaming execution, direct-manipulation gestures, ambient sensors without human gates) have no buffer for subtract to read. The pattern does not apply there.

## Where it applies

| Substrate | Buffer | Subtract gesture | Commit gesture |
|-----------|--------|-----------------|----------------|
| Shell (bash/zsh) | readline buffer | bound keystroke | enter |
| AAC tile grid | sentence strip | expand tile | speak tile |
| Voice (ASR) | interim transcript | wake phrase | silence/stop |
| Agent HTTP | request body | HTTP call | response handling |
| Switch/gaze/stylus | accumulating selection | explicit target | confirm action |

## Where it does not apply

- Streaming execution (live-eval, auto-run)
- Direct-manipulation gestures (swipe-to-delete, drag-to-resize)
- Ambient automation without human gate
- Interfaces that commit without intermediate representation

## The line

Subtract is invoked, never imposed. The user's existing competence is untouched. The translation is a subtraction, not a takeover. A user who never asks has a substrate bit-identical to one without subtract installed.

That property is verifiable, not aspirational.
