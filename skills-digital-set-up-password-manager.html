SET UP A PASSWORD MANAGER

Steps:
1. Choose a password manager. Reputable options: Bitwarden (free, open source), 1Password, KeePassXC (offline/local only).
2. Install the app on your phone and the browser extension on your computer.
3. Create your master password. Make it long (4+ random words, e.g., "correct horse battery staple"). This is the ONE password you must memorize. Write it down and store it somewhere physically secure (not on your computer).
4. Enable two-factor authentication on your password manager account.
5. Start adding accounts. Begin with the critical ones: email, banking, social media.
6. For each account: log in, let the manager save the credentials, then change the password to a randomly generated one (20+ characters). The manager remembers it — you don't have to.
7. Work through your remaining accounts over the next few weeks. No need to do all at once.

Notes:
- Your email password is the most important one. If someone has your email, they can reset every other account. Change it first.
- Do NOT store your master password in the password manager.
- If the password manager offers an emergency access feature, set it up with someone you trust.
- Export your vault periodically as an encrypted backup.
