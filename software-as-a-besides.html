SOFTWARE AS A BESIDES

       Type. The computer shows.

       Next time, type the command. Or don't — speak it. The software
       translates intent but does not assume.

       The software teaches to not need software to use a computer.

WHAT STAYS WHEN SUBTRACT IS GONE

       without translation:

         intent --> shell --> kernel

       with translation:

         intent --> subtract --> shell --> kernel
                     |
                     +-- routes intent to knowledge through lookdown.tsv

WHY "BESIDES"

       Software as a Service provides revenue abstracted away from intent.
       Software as a Besides brings human intent to the computer you have.

       One of these is bought once — or never.
