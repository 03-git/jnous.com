BROWSER PRIVACY SETTINGS

Steps:
1. FIREFOX (recommended for privacy):
   - Settings > Privacy & Security > Enhanced Tracking Protection: set to Strict.
   - Enable "Delete cookies and site data when Firefox is closed" (or manage exceptions for sites you want to stay logged into).
   - Set "Send websites a Do Not Track request": Always.
   - Disable "Allow Firefox to make personalized extension recommendations."

2. CHROME:
   - Settings > Privacy and Security > Third-party cookies: Block third-party cookies.
   - Settings > Privacy and Security > Ad privacy: turn off Topics, Site-suggested ads, and Ad measurement.
   - Settings > Sync: review what you're syncing to Google.

3. SAFARI:
   - Settings > Privacy: enable "Prevent cross-site tracking."
   - Enable "Hide IP address from trackers."

4. ALL BROWSERS:
   - Install uBlock Origin (not available on Chrome mobile — another reason to use Firefox).
   - Set your default search engine to DuckDuckGo or Brave Search for less tracking.
   - Disable autofill for payment methods and addresses in the browser — use your password manager instead.
   - Review installed extensions. Remove any you don't actively use.

Notes:
- "Incognito/Private mode" does not make you anonymous. It only prevents local history storage. Your ISP and websites still see you.
- For stronger privacy: use Firefox with uBlock Origin and consider a VPN from a reputable no-logs provider.
