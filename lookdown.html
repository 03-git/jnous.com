LOOKDOWN

       Intent defined produces forever outcomes.

       Lookdown is a tab-separated-value file. Each line is a pattern and
       a response. The computer matches top-to-bottom. First match wins.

WHERE ENTRIES COME FROM

       Computer answers intent through translation. Saves answers
       to the computer.

       Lookdown grows with intent. Shrinks through adaptation.
       Entries get pruned from the file and baked into the computer.

WHY IT SHRINKS

       Patterns match. Adapter learns patterns.
       Lookdown is pruned. Intent moves from file to the computer.

       The file gets smaller. The computer gets better. Both subtract
       to the human's loop.
