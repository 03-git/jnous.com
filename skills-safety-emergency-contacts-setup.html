SETTING UP EMERGENCY CONTACTS

Steps:
1. Create a file: echo "Emergency Contacts" > ~/emergency.txt
2. Add entries: echo "Name: ___ Phone: ___" >> ~/emergency.txt
3. Make it easy to find: ln -s ~/emergency.txt ~/Desktop/EMERGENCY.txt

What to include:
- Primary contact (family/friend)
- Doctor/clinic
- Local emergency number (911 in US)
- Poison control: 1-800-222-1222 (US)
- Insurance info if relevant

Notes:
- This file is local. It does not leave your device.
- Print a copy and keep it near the device.
